## chap09、顺序容器  291（317/864）

+ 一个**容器**就是一些特定类型对象的集合。**顺序容器（sequential container）的顺序不依赖于元素的值，而是与元素加入容器时的位置相对应**。

### 9.1、顺序容器概述

+ 所有顺序容器都提供了快速顺序访问元素的能力。但以下方面都有不同的性能折中：
  + 向容器添加或从容器中删除元素的代价
  + 非顺序访问容器中元素的代价
+ 顺序容器类型
  + vector
  + deque
  + list
  + forward_list
  + array
  + string
+ 确定使用哪种顺序容器
  + 除非你有很好的理由选择其他容器，否则应使用vector。
  + 如果程序要求随机访问元素，应使用vector或deque。
